# Aplicación RegistroEmpleados - Android (Jetpack Compose)

## Estructura del Proyecto

El proyecto está estructurado siguiendo las convenciones de desarrollo en Android con Kotlin:

```text
app/src/main/java/com/example/registroempleados/
│
├── Empleado.kt              # Modelo de datos de la entidad Empleado.
│
├── ui/theme/
│   └── Theme.kt             # Configuración de Material Theme 3 (Soporte Claro y Oscuro).
│
└── MainActivity.kt          # Actividad principal. Contiene la interfaz de usuario de 
                             Jetpack Compose y los logs del ciclo de vida.
```

## Cómo Ejecutar la Aplicación
Requisitos Previos:
- Android Studio (versión Ladybug o superior recomendada).
- JDK 17 o superior.
- Dispositivo físico Android con depuración USB activa o un Emulador (AVD) configurado con API 30 o superior.

## Pasos para Ejecución:
1. Clonar el repositorio: git clone https://github.com/JuanPablo88/iapdm-oto-2026-JuanDuarte.git
   - Abrir el proyecto
   - Inicia Android Studio, selecciona Open y busca la carpeta raíz del proyecto clonado.

4. Sincronizar Gradle:
- Espera a que termine el proceso de Gradle Sync.

5. Ejecutar:
- Selecciona tu dispositivo/emulador en la barra superior de Android Studio y presiona el botón verde de Run (Shift + F10).
